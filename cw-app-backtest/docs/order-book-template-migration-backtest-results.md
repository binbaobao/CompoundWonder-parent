## Model 1 差异说明

- 基准 Run 44：221 个样本，61 笔真实成交，闭仓胜率 60.655738%，平均收益 5.116073%，真实成交封板率 88.524590%。
- 迁移 Run 50：模板禁止二板加速后的集合竞价买入，并将连续竞价最早时间限制到 09:35；因此真实成交减少 1 笔。
- Run 50 新增的第 222 个样本是 603787 在 2025-06-12 的二板加速机会被阻止后，自动追加的 2025-06-13“3 进 4”观察样本；当天未触板，未成交。
- 固定候选逐笔对照无缺失；除模板阻断原因、上述递延机会，以及已存在的递延样本 600962 卖出生命周期变化外，其余业务结果保持一致。

## Model 3（2025）差异说明

- Run 53 与基准 Run 49 的样本数、成交数、闭仓数、胜率、平均收益、真实成交封板率及其他汇总指标完全一致。
- 356 个样本逐笔对照只有 603059（2025-08-01 买入）的卖出时间标记不同：规则 112 的条件、卖出日、卖出价 49.47 元和收益 46.447602% 均相同，时间由 09:30:58.670 变为 09:31:04.000。

## Model 2 差异说明

- 基准 Run 46：554 个样本，53 笔真实成交，闭仓胜率 66.037736%，平均收益 10.126512%，真实成交封板率 94.339623%。
- 迁移 Run 55：554 个样本，43 笔真实成交，闭仓胜率 60.465116%，平均收益 7.848465%，真实成交封板率 88.372093%。
- 普通首板启动市值达到 16 亿后禁止集合竞价买入：14 笔原竞价成交因盘中没有后续买点而转为虚拟持仓；4 笔原未成交样本在盘中回封时转为真实成交，净减少 10 笔真实成交。
- 002172 从 09:24:42 的集合竞价规则 6 改为 11:03:39 的连续竞价规则 13，买卖价格和收益不变。
- 其余同持仓类型样本中，56 笔只更新了模板阻断原因；600749、601566、600683 的卖出规则、日期、价格和收益均不变，仅触发时间标记有数秒漂移。
