# 回测订单簿会话重构基线

## 目标

本轮只调整回测订单簿与交易执行器的组织方式，不重新选股、不优化买卖参数，也不接入实盘交易。

- 历史静态事实迁移到 `TradeStaticFacts`。
- 证券与交易日机械参数迁移到 `MarketSessionSpec`。
- `OrderBook` 的正式回测构造只保留逐笔回放需要的热盘口。
- 初始化时按模式、昨日板高和启动市值预编译 `TradeExecutionTemplate`。
- 高频 Handler 只执行上海开盘集合竞价买、深圳开盘集合竞价买、连续竞价买、连续竞价卖、均价卖和尾盘集合竞价卖六个阶段。
- 固定复用既有任务的全部候选，比较重构前后结果。

## 代码与参数

- Model 1：`relay-model-006`
- Model 2：`multi-model-009`
- Model 3：`multi-model-009`
- 本轮没有修改选股、成交判定或买卖规则参数。
