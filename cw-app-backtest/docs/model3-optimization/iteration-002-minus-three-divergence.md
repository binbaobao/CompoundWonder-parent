# Model 3 优化迭代 002：2 进 3 小分歧阈值 -3%

## 结论

拒绝本轮策略参数，不作为下一版基准。

`-3%` 阈值提高了右尾收益和 2026 验证集结果，但使大量原本盈利退出的样本拖到亏损，2025 训练集及合并胜率显著下降，不符合“高收益同时保持较高胜率”的目标。

## 单变量参数

- 策略版本：`iteration-002`
- 固定源任务：6（2025）、4（2026-01-01 至 2026-07-21）
- 回放任务：13（2025）、14（2026）
- 唯一变化：`TwoToThreeSmallCapSellStrategy` 的连续三分钟价格/均价走弱，只有当前涨幅小于等于 `-3%` 才卖出
- 深度低开、买入日炸板次日开盘卖出以及其他板位规则全部保持不变
- 买入快照、虚拟持仓和样本集合与 iteration-001 一致

## 下一轮假设

问题不是“小分歧不能等”，而是统一等到亏损 `-3%` 才退出过于宽松。iteration-003 改为保护已有利润：连续走弱时，涨幅高于 `+2%` 先观察；若回落到 `+2%` 则卖出。这样强票仍有机会先封板，普通票则尽量在盈利状态退出。
