# Model 3 优化迭代 003：2 进 3 的 +2% 利润保护线

## 结论

保留本轮参数，作为新的策略基准。

连续三分钟价格或均价走弱时，涨幅高于 `+2%` 先观察，回落到 `+2%` 或以下立即卖出。该参数在 2025 训练集和 2026 验证集都提高了平均收益，真实、虚拟和全部场景胜率均与 iteration-001 基线完全一致。

## 单变量参数

- 策略版本：`iteration-003`
- 固定源任务：6（2025）、4（2026-01-01 至 2026-07-21）
- 回放任务：15（2025）、16（2026）
- 唯一变化：`TwoToThreeSmallCapSellStrategy` 连续走弱的利润保护阈值为 `+2%`
- 深度低开、买入日炸板次日开盘卖出和其他板位规则保持不变
- 样本：183 个触板样本，76 个真实成交，107 个虚拟场景，182 个已关闭，0 个错误
