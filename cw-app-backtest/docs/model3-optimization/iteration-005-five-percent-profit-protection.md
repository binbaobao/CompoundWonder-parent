# Model 3 优化迭代 005：2 进 3 的 +5% 利润保护线

## 结论

保留本轮参数，替代 iteration-004 的 `+4%` 基准。

将连续三分钟价格或均价走弱的利润保护线提高到 `+5%` 后，2025 真实、虚拟和全部场景平均收益均继续提高；2026 验证集完全不变。胜率、平均最大回撤和大亏样本数也没有恶化。

## 单变量参数

- 策略版本：`iteration-005`
- 固定源任务：6（2025）、4（2026-01-01 至 2026-07-21）
- 回放任务：19（2025）、20（2026）
- 唯一变化：`TwoToThreeSmallCapSellStrategy` 利润保护阈值由 `+4%` 提高到 `+5%`
- 样本：183 个触板样本，76 个真实成交，107 个虚拟场景，182 个已关闭，0 个错误
- 本轮没有重新选股，买入日期、时间、价格和规则均复用固定源任务
