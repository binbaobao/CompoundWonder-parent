# Model 1 单模式全量回测：relay-model-006

## 本轮目的

固定复用基线 Run 38 的 182 个候选，不重新选股。完善二板加速后的接力路径：三板一字板或高开秒板不买；如果三板最终封住，则在下一交易日把同一只股票作为独立的 3 进 4 机会，重新执行现有买入和卖出规则。

Model 2、Model 3、选股结果和卖出规则不变。

## 关键参数

- 有效回测任务：`runId=44`；
- 源任务：`sourceRunId=38`；
- 策略版本：`relay-model-006`；
- 交易模式：`tradeMode=1`，连板接力；
- 日期范围：`2025-01-01` 至 `2026-07-21`；
- 执行时间：`2026-07-22 22:09:12` 至 `2026-07-22 22:22:39`；
- 固定候选：182 笔，沿用 Run 38 已完成的连续交易日重复去重结果；
- 二板加速：二板为一字板，或振幅严格 `<3%`，或 T 字板且换手率严格 `<18%`，或换手率严格 `<15%`；
- 三板不买：三板为一字板，或开盘涨幅严格 `>4.5%` 且当日第一笔真实涨停价成交时间严格早于 `09:35:00`；
- “秒板”时间取 Level2 中 `09:30` 后第一笔涨停价成交，不使用 `09:15` 隔夜委托时间；
- 三进四重开：二进三未真实成交、三板收盘封住（`klineState=1..5`）时，以下一交易日建立 `selectionBoard=3`、`selectionTrigger=DEFERRED_3_TO_4` 的独立样本；
- 同一任务按 `symbol|tradeDate` 去重，既有固定候选优先，不重复创建同股票同日机会。

## 目标场景影响

### 三板拦截

原来会真实成交的 29 笔二进三被本轮规则拦截：

| 拦截原因 | 笔数 |
|---|---:|
| 二板加速后的三板一字板 | 15 |
| 二板加速后的三板高开秒板 | 14 |

这 29 笔在 Run 42 中有 17 笔盈利，平均收益 5.206259%。其中 25 笔三板最终封住，4 笔三板炸板。

4 笔炸板全部亏损，且因为三板没有封住，不会生成三进四机会：

| 股票 | 原买入日 | 买入日状态 | Run 42 收益 |
|---|---|---:|---:|
| 开创国际（600097） | 2025-03-24 | 11 | -20.232558% |
| 赛福天（603028） | 2025-03-24 | 12 | -12.800875% |
| 云煤能源（600792） | 2025-06-17 | 12 | -19.239374% |
| 中央商场（600280） | 2025-11-20 | 13 | -24.683544% |

4 笔平均亏损为 -19.239088%，新规则全部避开。

### 三进四重新买入

三板封住后共生成 39 个三进四样本：25 个来自本轮主动放弃的三板成交，另外 14 个来自此前本来就没有真实成交、但三板封住的样本。

| 指标 | 三进四派生样本 |
|---|---:|
| 全部机会 | 39 |
| 实际成交 | 19 |
| 虚拟卖出 | 11 |
| 未形成买卖样本 | 9 |
| 四板触板 / 封板 / 炸板 | 30 / 25 / 5 |
| 实际成交封板 | 17 / 19，89.473684% |
| 实际盈利 / 胜率 | 12 / 19，63.157895% |
| 实际平均收益 | 3.778690% |

19 笔真实三进四中，14 笔来自原来会在三板成交的股票，胜率 57.142857%、平均收益 2.336600%；另有 5 笔来自原来三板未成交的股票，胜率 80.000000%、平均收益 7.816542%。这证明“二进三没买到，三进四重新给机会”可以补回真实成交，但不能把延后一天视为无成本替换。

## 逐笔一致性

- 182 个固定候选中，29 个命中本轮三板拦截规则；
- 除这 29 个目标样本外，其余 153 个样本的持仓类型、买入日期/时间/价格/规则、卖出日期/时间/价格/规则、收益、最高封板和触板高度逐笔一致；
- Run 44 共 221 个样本，61 笔真实成交全部平仓，异常样本为 0。

## 失败任务说明

第一次执行生成了 Run 43，但在第 5 个样本首次写入派生机会时发现既有数据库 `selection_trigger` 字段长度不足以容纳原标识 `DEFERRED_THIRD_BOARD_CONFIRMATION`，任务因数据截断失败，没有形成可用策略结果。随后把标识缩短为 `DEFERRED_3_TO_4`，增加长度兼容回归断言并重新构建；Run 44 是本轮唯一有效的完整回测。

## 结论

本轮完整实现了目标路径，并且影响域准确：危险的三板加速炸板 4 笔全部避开，三板封住后产生 39 个三进四机会，其中 19 笔重新真实成交。实际成交封板率提高 1.200646 个百分点，平仓胜率提高 1.500808 个百分点。

代价同样明确：被延后的 25 个封住三板在原策略中平均收益为 9.117515%，更晚的三进四买点会少吃一个涨停板，并可能直接承受四板分歧，因此总真实成交减少 10 笔、平均收益下降 0.394728 个百分点。该规则应定位为降低三板加速炸板大亏风险、保留后续重新参与权的风险控制规则，而不是单纯提高收益的规则。
