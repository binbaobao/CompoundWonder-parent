# Model 1 单模式全量回测：relay-model-001

## 本轮目的

建立连板接力模式的固定候选基线。回测覆盖 2025-01-01 至 2026-07-21，所有候选独立执行买卖，不受仓位和候选重叠影响。

同一股票连续多个交易日重复入选时，整条连续链只保留第一笔；中间断开一个交易日后再次入选，重新作为独立样本。后续参数迭代固定复用本轮 182 个候选，不再重新选股。

## 关键参数

- 回测任务：`runId=38`；
- 策略版本：`relay-model-001`；
- 交易模式：`tradeMode=1`，连板接力；
- 日期范围：`2025-01-01` 至 `2026-07-21`；
- 执行时间：`2026-07-22 19:19:34` 至 `2026-07-22 19:47:52`；
- 选股版本：`relay-v1`；
- 选股强度：沿用当前触发场景产生的 STRICT、NORMAL、RELAXED，不增加强度级别二次过滤；
- 弱唯一 5 板：沿用触发日前 10 个交易日平均高度严格 `<6` 的门槛；
- 候选上限：各触发场景沿用当前 Top3；
- 连续日去重：217 笔原始入选中删除 35 笔连续交易日重复样本，保留 182 笔；
- 买入：沿用当前 Model 1 集合竞价、隔夜委托和 14:30 前连续竞价规则；
- 卖出：沿用当前按板位和启动市值分发的卖出引擎。
