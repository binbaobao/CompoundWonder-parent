# Model 1 单模式全量回测：relay-model-002

## 本轮目的

固定复用基线 Run 38 的 182 个候选，不重新选股，只验证深圳集合竞价规则 7 的最低分歧门槛：竞价全价位卖量必须达到涨停买量的 20%，并继续满足原有严格小于 40% 的上限。

规则 6 的竞价大单路径、撤单公式、其他买入规则和全部卖出规则均不调整。

## 关键参数

- 回测任务：`runId=39`；
- 源任务：`sourceRunId=38`；
- 策略版本：`relay-model-002`；
- 交易模式：`tradeMode=1`，连板接力；
- 日期范围：`2025-01-01` 至 `2026-07-21`；
- 执行时间：`2026-07-22 19:55:18` 至 `2026-07-22 20:05:27`；
- 固定候选：182 笔，沿用 Run 38 已完成的连续交易日重复去重结果；
- 选股强度：不区分过滤 STRICT、NORMAL、RELAXED；
- 本轮唯一参数：规则 7 的 `全价位卖量 / 涨停买量` 从仅 `<40%` 改为 `[20%, 40%)`。

## 参数实际影响

规则 7 的最低分歧门槛确实生效：

- 阳光乳业（001318，2025-03-19）：卖量占买量 33.24%，仍由规则 7 买入，封板，收益 1.801802%；
- 环球印务（002799，2025-08-04）：14.47%，规则 7 被拦截，但 09:20:43 又由规则 6 买入，炸板，收益 -19.320215%；
- 中天服务（002188，2025-12-22）：17.31%，规则 7 被拦截，但 09:23:32 又由规则 6 买入，炸板，收益 -22.992701%。

因此规则 7 的成交从 3 笔降为 1 笔，但两笔被拦截样本转入规则 6，最终成交数量、封板率、胜率和收益全部不变。

## 结论与下一轮

本轮为负实验：参数本身有效，但没有改变最终买卖结果。根因不是 20% 阈值无效，而是同一竞价阶段的规则 6 可以绕过规则 7 的最低分歧门槛。

`relay-model-003` 继续固定复用 Run 38 的 182 个候选，保留规则 7 的 `[20%, 40%)` 区间，并验证规则 6 回落路径是否也应拒绝卖量占买量低于 20% 的无换手盘口；其他规则保持不变。
