# 连板接力选股模式 V1

> 业务状态：已确认
> 确认日期：2026-07-22
> 实现状态：待与当前代码对齐
> 范围：只定义连板接力的选股触发、候选板数、选股强度、排序和数量，不定义买入、卖出和持仓执行规则。

## 1. 文档定位

本文是后续连板接力选股改造和分场景回测的目标业务口径。

`stock-selection-logic.md` 仍用于描述当前工作区实际实现；在代码完成对齐之前，不得把本文规则描述成已经上线的行为。后续实现、回测或页面展示涉及连板接力模式时，应先读取本文，再检查生产代码与本文的差异。

## 2. 核心概念

### 2.1 高度身份

- 股票达到 7 板，称为龙头。
- 股票达到 9 板，称为妖股。
- 股票达到 11 板，称为大帝。
- 股票从 7 板开始占领情绪周期。

### 2.2 情绪周期占领

股票达到连板高度 `H` 后，占领数量为：

```text
占领数量 = H - 3
```

| 已完成连板高度 | 占领数量 |
|---:|---:|
| 7 板 | 4 个 |
| 8 板 | 5 个 |
| 9 板 | 6 个 |
| 10 板 | 7 个 |
| 11 板 | 8 个 |

后续股票没有超过当前占领者的历史最高连板高度时，只能算跟风，跟风股断板不能触发高切低。

新股票在当日收盘封死的连板高度超过当前占领者的历史最高连板高度时，当天立即接管情绪周期：

```text
新股票当日封死连板高度 > 当前占领者历史最高连板高度
```

例如，前一占领者最高完成 7 板并在 7 进 8 时断板，后续股票完成 7 板时仍然是跟风；后续股票封死 8 板时，当天立即接管情绪周期。

### 2.3 断板定义

断板只看收盘是否涨停：

- 盘中炸板但收盘重新封死涨停，不算断板；
- 收盘没有封住涨停，算断板。

只有当前情绪周期占领者断板，才触发高切低。

## 3. 触发体系与优先级

连板接力分为三类触发：

1. 高度压制；
2. 高切低；
3. 卡位。

主选股链使用互斥的 `if / else if`；卡位在主链之后单独判断：

```java
if (命中高度压制) {
    // 高度压制选股
} else if (命中高切低第二次补充) {
    // 先完成前一日已经确认的断板事件
} else if (命中高切低断板当天) {
    // 再判断当日是否发生新的占领者断板
}

if (前面没有选出候选 && 命中弱唯一5板) {
    // 最后的卡位选股
}
```

主链顺序必须保持为“高度压制 -> 高切低第二次补充 -> 高切低断板当天”。断板第二次补充放在断板当天之前，是为了避免连续退潮时两个高度条件同时成立，从而把同一次断板重复识别成新的断板。

## 4. 高度压制

### 4.1 触发条件

只要当日市场最高连板高度为 2、3、4 板，就属于高度压制，不要求相对前一交易日下降。

以下情况都属于高度压制：

- 昨日 3 板，今日仍为 3 板；
- 昨日 2 板，今日升到 3 板；
- 昨日 4 板，今日降到 3 板；
- 昨日 3 板，今日升到 4 板。

当日市场最高板达到 5 板及以上时，才退出高度压制分支。

### 4.2 候选板数、强度和数量

| 当日市场最高板 | 候选板数 | 选股强度 | 数量上限 |
|---:|---:|---|---:|
| 2 板 | 2 板 | 宽松 | 3 只 |
| 3 板 | 3 板 | 宽松 | 与 2 板合计 3 只 |
| 3 板 | 2 板 | 一般 | 与 3 板合计 3 只 |
| 4 板 | 3 板 | 宽松 | 与 2 板合计 3 只 |
| 4 板 | 2 板 | 一般 | 与 3 板合计 3 只 |

## 5. 高切低

### 5.1 占领者断板当天

当前情绪周期占领者在当日收盘没有封住涨停时，触发第一次高切低选股。

| 候选板数 | 选股强度 | 数量上限 |
|---:|---|---:|
| 3 板 | 严谨 | 与 2 板合计 3 只 |
| 2 板 | 严谨 | 与 3 板合计 3 只 |

时间口径：

```text
D 日收盘确认占领者断板
-> D 日收盘执行第一次高切低选股
-> 候选用于 D+1 交易
```

### 5.2 断板后的第二次补充

第二次补充的触发条件为：

```text
前一交易日发生情绪周期占领者断板
并且
当前交易日的情绪周期占领代码没有更换
```

| 候选板数 | 选股强度 | 数量上限 |
|---:|---|---:|
| 3 板 | 一般 | 3 只 |

时间口径：

```text
D 日收盘：占领者断板，执行第一次高切低选股
D+1 收盘：执行第二次补充选股，候选用于 D+2 交易
```

第二次补充直接执行选股，不以前一次是否已经买入作为前置条件。前一次没有买到，或者买入后炸板并快速卖出，属于后续买卖执行规则，不改变第二次选股本身。

如果 D+1 有新股票超过前占领者的历史最高高度并封死涨停，情绪周期占领代码发生变化，则取消第二次补充。新的占领者说明市场接力情绪良好、没有产生高切低所需要的分歧。

## 6. 卡位

卡位是主选股链之后的独立兜底判断。

### 6.1 触发前提

必须同时满足：

1. 主选股链没有产生候选；
2. 当日市场最高板为 5 板；
3. 过滤 ST 后恰好只有一只 5 板股票；
4. 唯一 5 板的质量指标完整；
5. 唯一 5 板命中至少一项弱势条件。

### 6.2 弱唯一 5 板条件

以下任意一项成立即视为弱：

| 指标 | 触发条件 |
|---|---:|
| 当日流通市值 | `>45亿元` |
| 当日换手率 | `>45%` |
| 当日振幅过小 | `<2.5%` |
| 当日振幅过大 | `>13%` |
| 本轮启动价格过低 | `<2.5元` |
| 本轮启动价格过高 | `>30元` |

边界值 45 亿元、45%、2.5%、13%、2.5 元和 30 元本身不触发弱势判断。

### 6.3 卡位选股

| 候选板数 | 选股强度 | 数量上限 |
|---:|---|---:|
| 2 板 | 严谨 | 3 只 |

## 7. 三级选股强度

三种强度使用同一套评分公式。强度只影响过滤条件和阈值，评分只负责同板候选排序，不设最低评分门槛。

| 条件 | 严谨 | 一般 | 宽松 |
|---|---:|---:|---:|
| 启动流通市值 | `<20亿元` | `<30亿元` | `<50亿元` |
| 最近 200 根有效 K 线历史最大换手率 | `<=40%` | `<=45%` | `<=50%` |
| 启动价格 | `>3元` | 不限制 | 不限制 |
| 最低评分 | 不设 | 不设 | 不设 |
| 完整筹码判断 | 执行 | 执行 | 执行 |
| 其他共同过滤 | 暂时沿用现有规则 | 暂时沿用现有规则 | 暂时沿用现有规则 |

启动流通市值和启动价格沿用现有口径，取本轮首板前一个交易日收盘时的数据。历史最大换手率取本轮启动前最近 200 根有效日 K 的最大换手率。

V1 阶段除表中三项差异外，前 20 日异常、加速缩量板、近期形态、90 日历史指标以及其余完整筹码判断暂时使用同一套现有规则。先按场景和强度完成回测，再根据结果调整。

## 8. 排序与数量截断

所有同时包含 2 板和 3 板候选的场景统一执行：

1. 连板高度降序，3 板优先于 2 板；
2. 同板内按选股分数降序；
3. 同板同分时，按选股当日价格升序；
4. 价格仍相同时，按股票代码升序；
5. 最终截取前 3 只。

如果合格 3 板票已经达到 3 只，则全部选择 3 板票，2 板票不再入选；3 板不足 3 只时，才使用 2 板票补足。

## 9. V1 回测原则

- 每条选股结果必须能够区分触发类型：高度压制、高切低当天、高切低第二次补充、弱唯一 5 板卡位；
- 每条结果必须能够区分选股强度：严谨、一般、宽松；
- 先保持其他共同过滤不变，避免一次回测同时改变过多维度；
- 后续根据各触发点、板数和强度的独立回测结果调整规则；
- 在业务规则再次确认前，不把买入、卖出和持仓状态混入本文。
