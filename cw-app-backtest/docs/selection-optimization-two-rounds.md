# 选股放宽两轮全量回测

## 1. 固定实验口径

- 回测区间：2025-01-01 至最新完整交易日 2026-07-22。
- 固定基准：run 162。
- 第一轮只增加“三板加速缩量”备用候选，不改变原主候选 Top3 的入选和顺序。
- 第二轮在第一轮基础上，只把普通首板当前流通市值上限从 22 亿元放宽到 24 亿元。
- 两轮都使用当前统一买卖执行链路；不增加卖出分发器，不修改卖出规则。
- 每轮验收期末资产、复利收益、最大回撤、交易数、胜率、资金利用率和全部交易路径变化。

## 3. 第一轮：三板加速缩量备用候选

### 3.1 规则边界

- 原有主候选仍按既定规则筛选、排序并占据 Top3。
- 只有被“三板加速缩量”这一条规则拒绝、但通过其余强度规则的候选，才可进入备用池。
- 备用候选只补主候选不足 3 个的空位，不能挤掉任何原主候选。
- 回测买入时先遍历全部普通候选；普通候选均未成交且账户空仓时，才尝试备用候选。
- 当日卖出后尝试备用候选时，沿用卖出时刻之后买入和排除刚卖股票的约束。

第一轮完整回测为 run 163，状态成功，完整结束于 2026-07-22。

run 163 确实提高了操作频率和资金利用率，但新增样本质量很差，收益与回撤都显著恶化。因此“更多操作机会”本身不能作为放宽条件成立的证据。

第一轮在这个短区间内新增大胜达和宜宾纸业，两笔都盈利，使同区间复利提高约 10.11%。但扩展到完整 19 个月后结论完全相反，证明不能用 run 76 短区间的两只正样本决定全年规则。

备用候选进一步造成 2 笔非备用主候选新增成交：

- 神力股份：2025-03-14 至 03-17，-17.92%。
- 康惠制药：2025-03-18 至 03-24，+20.55%。

相对 run 162 被资金路径挤掉的 4 笔是：

- 德联集团：-5.60%。
- 奇精机械：+46.42%。
- 联合精密：-5.14%。
- 键邦股份：-19.99%。

其中最关键的是 2025 年 3 月替代链。run 162 的“奇精机械、联合精密、键邦股份”合计复利 1.1112 倍；run 163 的“园林股份、神力股份、康惠制药、太阳电缆”合计仅 0.8322 倍，这一段相对损失约 25.11%。虽然太阳电缆规避了键邦股份的大亏，但远不足以补偿错过奇精机械。

其余 81 笔共同交易的买入日、买入时刻、卖出日和卖出时刻一致。少量收益率末位差异来自资金规模变化后的股数与手续费取整，不属于买卖规则变化。

第一轮结论：三板加速缩量条件不能整体作为备用池放开。它提高资金利用率，却引入一个只有 30.77% 胜率、复利亏损 54.99% 的低质量样本组。若继续研究，只能从新通联、香江控股、宜宾纸业三个正样本反推更窄的事前可观察条件，不能保留当前整组放宽。
