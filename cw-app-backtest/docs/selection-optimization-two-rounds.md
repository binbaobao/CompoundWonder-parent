# 选股放宽两轮全量回测

## 1. 固定实验口径

- 回测区间：2025-01-01 至最新完整交易日 2026-07-22。
- 固定基准：run 162。
- 第一轮只增加“三板加速缩量”备用候选，不改变原主候选 Top3 的入选和顺序。
- 第二轮在第一轮基础上，只把普通首板当前流通市值上限从 22 亿元放宽到 24 亿元。
- 两轮都使用当前统一买卖执行链路；不增加卖出分发器，不修改卖出规则。
- 每轮验收期末资产、复利收益、最大回撤、交易数、胜率、资金利用率和全部交易路径变化。

## 3. 第一轮：三板加速缩量备用候选

### 3.1 规则边界

- 原有主候选仍按既定规则筛选、排序并占据 Top3。
- 只有被“三板加速缩量”这一条规则拒绝、但通过其余强度规则的候选，才可进入备用池。
- 备用候选只补主候选不足 3 个的空位，不能挤掉任何原主候选。
- 回测买入时先遍历全部普通候选；普通候选均未成交且账户空仓时，才尝试备用候选。
- 当日卖出后尝试备用候选时，沿用卖出时刻之后买入和排除刚卖股票的约束。

第一轮结论：三板加速缩量条件不能整体作为备用池放开。它提高资金利用率，却引入一个只有 30.77% 胜率、复利亏损 54.99% 的低质量样本组。若继续研究，只能从新通联、香江控股、宜宾纸业三个正样本反推更窄的事前可观察条件，不能保留当前整组放宽。

## 4. 第二轮：普通首板流通市值上限 22 亿放宽至 24 亿

### 4.1 规则边界

- 普通首板当前流通市值保持严格小于上限。
- 第一轮的备用候选逻辑保持不变。
- 除流通市值阈值外，不修改其他普通首板条件。

### 4.3 全部交易路径变化

流通市值放宽后，完整区间一共产生 65 个处于 22–24 亿元的普通首板盯盘任务。其中只有 4 个触发买入规则，触发率 6.15%；3 个实际成交，成交率 4.62%：

| 股票 | 选股日流通市值 | 买入规则结果 | 最终结果 |
|---|---:|---|---:|
| 时代出版 | 22.8251 亿元 | 2025-05-30 14:27:19，规则 14 成交 | -4.93% |
| 赛象科技 | 23.5636 亿元 | 2025-10-23 09:39:59，规则 13 成交 | -6.42% |
| 杭州解百 | 22.0070 亿元 | 2026-02-05 09:51:35.660 触发规则 14，但未成交 | 无持仓 |
| 大连热电 | 23.1560 亿元 | 2026-05-07 09:48:33，规则 14 成交 | +4.57% |

杭州解百的最后可成交委托时刻为 09:51:35.920。规则在 09:51:35.660 触发，但沪市回测延迟为 450ms，委托到达时已经晚于最后可成交时刻，因此没有形成持仓。这是执行时序未成交，不是资金占用。

时代出版改变了 2025-05-30 至 06-06 的资金链：

- run 163：劲旅环境 -17.48%，随后中衡设计 +4.25%，合计复利 0.8603 倍。
- run 164：时代出版 -4.93%，随后买入连板主候选共创草坪 +7.82%，合计复利 1.0251 倍。
- 新路径相对旧路径提高约 19.15%，是第二轮总收益改善的主要来源。

除这个资金替代链外：

- 赛象科技是独立新增亏损 -6.42%。
- 大连热电是独立新增盈利 +4.57%，没有挤掉 2026-05-13 的万控智造。
- 其余 94 笔共同交易的买卖日期和时刻一致；收益率末位差异来自本金、股数和手续费取整。
- 规则记录从 373 条增加到 376 条。除上述直接候选和资金链产生的规则外，没有其他成交路径变化。

第二轮结论：24 亿阈值在当前组合上带来 +16.73% 的期末资产改善，也正确补到了 run 76 中的大连热电；但直接新增三笔自身是亏损组，总改善依赖“用时代出版 -4.93% 替代劲旅环境 -17.48%，并衔接共创草坪 +7.82%”这一条资金路径。这个证据支持继续保留实验记录，不足以单独证明 24 亿上限具有稳定优势。

## 5. 两轮结论

两轮叠加后的 run 164 与原基准 run 162 对比：

|---|---:|---:|---:|
| 期末资产 | 25,910,454.21 | 12,706,205.40 | -13,204,248.81，-50.96% |
| 累计收益 | +25,810.45% | +12,606.21% | -13,204.25 个百分点 |
| 已平仓交易 | 85 | 98 | +13 |
| 盈利/亏损 | 53 / 32 | 58 / 40 | +5 / +8 |
| 胜率 | 62.35% | 59.18% | -3.17 个百分点 |
| 逐笔复利倍数 | 259.5755 倍 | 128.4746 倍 | -50.51% |
| 最大回撤 | 47.01% | 68.64% | 恶化 21.63 个百分点 |
| 活跃交易日 | 220 / 375 | 240 / 375 | +20 |
| 资金活跃率 | 58.67% | 64.00% | +5.33 个百分点 |
| 最长连续空仓 | 21 个交易日 | 17 个交易日 | -4 |

最终判断：

1. 三板加速缩量备用池不应按当前宽口径进入正式策略。它确实增加操作机会，但 13 笔直接成交只有 30.77% 胜率，导致组合期末资产相对 run 162 大幅下降。
2. 24 亿普通首板上限对 run 163 有正向净效应，并补到大连热电；但三笔直接样本自身为亏损，正向结果高度依赖资金替代链，证据强度不足。
3. run 164 的资金利用率提高到 64.00%，仍低于 run 76 同区间的 70.21%；同区间仍缺少海森药业、均瑶健康、苏州高新、越剑智能、宝鼎科技、华塑控股、海欣股份和哈药股份等 run 76 独有交易。继续机械放宽条件会重复第一轮的风险。
4. 若后续继续验证，优先方向不是再扩大候选池，而是撤掉第一轮宽口径备用池后，单独在 run 162 基线上验证 24 亿上限；这需要另一轮独立回测，本次按要求到两轮为止。
