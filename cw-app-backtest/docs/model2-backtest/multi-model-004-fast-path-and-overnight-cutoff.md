# Model 2 参数迭代：multi-model-004

## 目的

保持 run 24 的 554 个候选、选股结果和全部卖出规则不变，只验证真实成交当天炸板样本的两类买入侧特征：

1. 普通连续竞价规则 14：若已完成分钟收盘价首次达到 7%，不足 8 个交易分钟就触发买入，则拒绝该次信号；大单规则 11-13 不受影响，后续出现新买点仍可成交。
2. Model 2 隔夜单：计划拒绝队首时间为 0 或 14:30 以后才排到队首的模拟成交，不影响 Model 1/3。

连续竞价最低实时换手率仍保留 `multi-model-003` 的 12%。

## 失败原因

逐笔对比 run 25 与 run 26 后确认，共减少 17 笔真实成交：

- 连续竞价路径条件减少 3 笔：2 笔炸板亏损、1 笔封板亏损；该条件方向有效，且没有删掉盈利样本。
- 隔夜条件减少 14 笔：计划内的 4 笔均为队首时间为 0 或 14:30 后才排到队首的炸板亏损；另外误删 10 笔 `09:15:00.xxx` 队首时间样本，其中 9 笔盈利、1 笔亏损。

实现错误在于 Model 2 新方法复用了旧的 `lastOrderTime > 09:15:01.000` 前置条件。离线消融的真实意图是：只要存在真实队首时间且早于 14:30 就保留；不再使用成交额去兜底 `0` 或尾盘队首时间。
