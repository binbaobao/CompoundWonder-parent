# multi-model-009：普通首板卖出场景基线清理

## 目标与边界

以 Model2 单模式全量回测 Run 32 为行为基准，只清理普通市值首板的分板位卖出策略：

- 保留 Run 32 的 53 笔真实成交和 124 笔虚拟卖出实际使用的普通市值卖出场景；
- 删除在这 177 个卖出样本中没有使用的普通市值卖出分支，并为没有直接样本的板位保留经过其他板位真实样本确认的安全兜底；
- 每个保留分支增加代表样本注释，写明股票名称、代码、卖出日期和毫秒级时间；
- 不修改买入侧、小市值首板卖出、尾盘竞价规则 1、买入日炸板次日开盘规则 301，以及普通/小市值共用的节前规则 115。

## 结论

最终版本没有改变 Run 35 的任何真实买卖结果，同时修复了无直接样本板位盘口策略永久返回 `false` 的漏洞。当前普通首板卖出代码保留一年半固定候选回放实际命中的分板位场景，并通过共享兜底覆盖 4进5、8进9、9板以上；每条规则都可追溯到股票名称、代码、日期和时间。后续出现新样本时，应先用固定候选回放确认影响域，再补充或收窄对应板位规则。
