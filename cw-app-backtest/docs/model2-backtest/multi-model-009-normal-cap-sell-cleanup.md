## 目标与边界

以 Model2 单模式全量回测 Run 32 为行为基准，只清理普通市值首板的分板位卖出策略：

- 保留 Run 32 的 53 笔真实成交和 124 笔虚拟卖出实际使用的普通市值卖出场景；
- 删除在这 177 个卖出样本中没有使用的普通市值卖出分支；
- 每个保留分支增加代表样本注释，写明股票名称、代码、卖出日期和毫秒级时间；
