# Model 2 参数迭代：multi-model-002

## 本轮目的

验证普通首板以“启动流通市值不超过约 20 亿元”为上限时的选股和回测结果。
由于首板上涨约 10%，基础候选池改为首板当日流通市值严格 `<22亿元`。

## 唯一参数变化

| 参数 | multi-model-001 | multi-model-002 |
|---|---:|---:|
| Model 2 首板当日流通市值上限 | `<20亿元` | `<22亿元` |

边界口径：

- `219999.99万元`：进入基础候选池；
- `220000万元`：不进入基础候选池；
- Model 2 启动流通市值下限仍为 `>=119999万元`；
- 其他选股过滤、评分、Top3、买入和普通市值卖出规则全部保持不变；
- Model 1 和 Model 3 不受本轮参数影响。

## 基准与目标区间

- 对照基准：`multi-model-001`，`runId=23`
- 基准结果：493 个样本，58 笔实际成交，胜率 50.000000%，平均收益 4.471978%，异常 0
- 新回测区间：`2025-01-01` 至 `2026-07-21`
- 仓位口径：所有候选独立执行买卖，不受仓位和时间重叠限制
- 重点样本：600488 中铁工业，2026-03-27 首板当日流通市值约 20.75 亿元
