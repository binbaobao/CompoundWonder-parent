# Model 2 参数迭代：multi-model-002

## 本轮目的

验证普通首板以“启动流通市值不超过约 20 亿元”为上限时的选股和回测结果。
由于首板上涨约 10%，基础候选池改为首板当日流通市值严格 `<22亿元`。

## 唯一参数变化

| 参数 | multi-model-001 | multi-model-002 |
|---|---:|---:|
| Model 2 首板当日流通市值上限 | `<20亿元` | `<22亿元` |

边界口径：

- `219999.99万元`：进入基础候选池；
- `220000万元`：不进入基础候选池；
- Model 2 启动流通市值下限仍为 `>=119999万元`；
- Model 2 实际买入的启动流通市值上限仍严格 `<200000万元`，本轮没有放宽买入侧；
- 其他选股过滤、评分、Top3、买入和普通市值卖出规则全部保持不变；
- Model 1 和 Model 3 不受本轮参数影响。

## 结论

本轮只放宽 Model 2 基础候选池市值上限，样本数和实际成交数均增加；实际胜率保持不变，实际平均收益、合并场景胜率和合并场景平均收益均提高。首板触板后的整体封板率基本持平并小幅提高，但实际买入样本的入场封板率降低 `2.839757` 个百分点，因此本轮收益改善伴随一定的实际入场封板质量下降，后续优化应同时观察这两个口径。
