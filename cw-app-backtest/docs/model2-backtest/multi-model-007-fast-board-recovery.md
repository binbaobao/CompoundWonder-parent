# multi-model-007：快速冲板后的恢复条件实验

## 实验目标

固定复用 Run 28 的 554 只候选，不重新选股，在 `multi-model-006` 基础上修正两条买入旁路：

1. 上海竞价规则 2 原始绝对强度成立但启动流通市值 `>=160000万元` 时，同一盘口事件不允许降级为规则 3；纯封单增长信号仍可按规则 3 判断；
2. 已完成分钟从首次达到 7% 后不足 8 个交易分钟就封板时，后续连续竞价只有实时换手率 `>=25%` 才恢复买入。

卖出规则、候选、隔夜成交和其他买入参数不变。策略版本为 `multi-model-007`，代码提交为 `51d0be0`。

## 误伤原因与第三轮方向

分钟盘口对比显示：

- 津药药业当天约平开，先充分换手、炸开，再于 09:39 回封；
- 凯盛新能约高开 4.6%，广宇集团约高开 7.2%，神力股份约高开 9.2%，均属于高开后快速冲板再炸；
- 当前“快速冲板后换手不足 25% 一律拒绝”没有区分低位换手板与高开加速板，因此误删津药药业。

第三轮保持 16 亿元竞价边界和 25% 换手恢复条件不变，仅验证一个通用例外：发生过快速冲板后，如果当日开盘涨幅不高于 3%，允许按原有连续竞价规则再次判断；高开超过 3% 的快速冲板仍需换手达到 25% 才恢复。

## 结论

`multi-model-007` 证明了“快速冲板后不能无条件重新买回”能显著降低炸板率，但 25% 单一阈值过宽地过滤了低开换手型强票。该版本作为未定稿实验保留，不作为最终参数。
