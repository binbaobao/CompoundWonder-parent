# Model 2 参数迭代：multi-model-005

## 目的

修正 `multi-model-004` 的 Model 2 隔夜成交边界错误。只改一处：

```text
lastOrderTime > 0 && lastOrderTime < 14:30:00.000
```

含义是必须存在真实队首时间，且 14:30 前已经排到队首；不再使用当日成交额去兜底队首时间为 0 或尾盘才排到的样本。

以下内容全部保持不变：

- run 24 的 554 个固定候选，不重新选股；
- 普通连续竞价规则 14 的“完成分钟首次到 7% 后至少等待 8 个交易分钟”；
- 连续竞价最低实时换手率 12%；
- 大单规则 11-13；
- 全部卖出规则。

## 逐笔影响域

相对 run 25：

- 删除 7 笔旧成交：6 笔买入日炸板亏损，1 笔封板亏损；没有删除盈利样本。

  - 603700 宁水集团，收益 10.038610%。

## 结论与下一步

保留 `multi-model-005` 作为新的买入侧基线。按既定顺序，下一阶段不再叠加买入或选股条件，先分析 51 笔已封板真实成交中：

1. 下一交易日低开的样本；
2. 盘中没有正溢价或最终卖出亏损的样本；
3. 它们在首板日、买入日、竞价/连续竞价路径和现有卖出触发上的共同特征。

完成特征分层后再决定是否调整卖点，最后才做重新选股的全量影响域回测。
