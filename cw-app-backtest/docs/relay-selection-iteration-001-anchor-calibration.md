## 2. 参数演进

### 2.1 宽放对照 run 5

最初同时放开一般市值、2 板振幅、全部加速缩量否决和两条筹码通道。5 只锚点全部入选，但 Top3 质量明显下降：

| 指标 | run 5 |
|---|---:|
| 合格候选 | 340 |
| Top3 入选 | 310 |
| Top3 封板率 | 79.06% |
| Top3 理论胜率 | 72.49% |
| Top3 平均理论最大收益 | 15.54% |

因此 run 5 只作为“放宽过度”的对照，不作为最终参数。

### 2.2 3 板保护 run 6

恢复 3 板加速缩量否决，只让 2 板继续放宽。森林包装从宽放轮第 2 名恢复为第 1 名，5 只锚点仍全部入选。

run 6 共入选 263 只，Top3 封板率 82.80%、理论胜率 75.48%、平均理论最大收益 16.76%。质量较 run 5 回升，但仍低于基线封板率和胜率。

逐股对比确认，run 6 相对基线新增入选中，原来被完整筹码阶梯拒绝的 36 只封板率只有 75%、胜率 64.29%，是主要拖累来源。

### 2.3 收窄带状参数 run 7

基于 run 6 已落库的全候选，对 324 个仍能保留全部锚点的参数组合重新执行同样的 Top3 排序。按封板率优先，选定以下通用边界：

- 一般强度启动流通市值 `<35亿元`；
- 2 板 5 日复权振幅 `<35%`；
- 2 板出现两根加速缩量板时，历史最大换手率必须 `<20%`；
- 3 板出现两根加速缩量板时继续否决；
- 低价高换手筹码通道：启动流通市值 `>20亿元且<=30亿元`、历史最大换手率 `>=40%且<46%`、选股日价格 `<10元`；
- 中市值筹码通道：启动流通市值 `>25亿元且<=30亿元`、历史最大换手率 `>=25%且<30%`、选股日价格 `>=16元且<25元`。

这些通道不绕过 ST、可转债、涨幅、当日价格、强度市值/换手、前 20 日形态、90 日历史指标和其余完整筹码判断。
